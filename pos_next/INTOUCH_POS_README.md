# InTouch POS - نسخة الكاشير الجديدة

تم إنشاء نسخة جديدة من POS Next مستوحاة من تصميم برنامج الكاشير InTouch POS.

## 🎨 المميزات

### التصميم
- ✨ واجهة مستخدم حديثة وجذابة مستوحاة من InTouch POS
- 🎯 شبكة أزرار كبيرة وواضحة للمنتجات
- 📱 لوحة عرض تفصيلية على اليمين مع لوحة مفاتيح رقمية
- 🎨 نظام ألوان متناسق وجميل (أزرق سماوي، رمادي، أخضر، أحمر)
- ⚡ أزرار تحكم دائرية سريعة (HOLD، +، -، X)

### المكونات الرئيسية
1. **ItemsSelectorInTouch** - شبكة الأزرار الزرقاء الكبيرة للمنتجات
2. **InvoiceCartInTouch** - لوحة العرض اليمنى مع:
   - عرض رقم الطاولة والإجمالي
   - أزرار التحكم العلوية (Tables، Delivery، Hall، Take Away، etc.)
   - زر "End Bill" الكبير
   - 4 أزرار دائرية للتحكم السريع
   - قائمة العناصر المختارة
   - الإجماليات
   - لوحة المفاتيح الرقمية
3. **POSSaleInTouch** - الصفحة الرئيسية الجديدة

## 🚀 كيفية الوصول

### 1. الوصول عبر المتصفح
افتح المتصفح واذهب إلى:
```
http://your-site.com/pos/intouch
```

أو:
```
http://localhost:8000/pos/intouch
```

### 2. Build الأصول الجديدة
بعد التعديلات، تأكد من build الأصول:

```bash
cd /home/eslam/frappe-bench/apps/pos_next/POS
yarn build
```

ثم:
```bash
cd /home/eslam/frappe-bench
bench --site your-site-name build --app pos_next
bench restart
```
